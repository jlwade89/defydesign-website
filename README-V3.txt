THIS PACKAGE IS READY FOR GITHUB PAGES.

UPLOAD/REPLACE THESE FILES IN YOUR EXISTING REPOSITORY:
- index.html
- styles.css
- script.js
- CNAME
- robots.txt
- sitemap.xml
- assets/favicon.svg
- assets/og-card.svg

EASIEST UPDATE METHOD
1. Open your GitHub repository.
2. Delete or overwrite the existing site files.
3. Upload every file and the assets folder from this package.
4. Click Commit changes.
5. GitHub Pages will redeploy automatically.

DOMAIN
The CNAME file is already set to:
defydesign.store

IMPORTANT BEFORE TAKING REAL ORDERS

1. EMAIL
The website currently uses:
orders@defydesign.store

That mailbox must exist through Namecheap Private Email, Google Workspace, Microsoft 365, or another provider.

2. PRICES
The prices are placeholders inside script.js:
50 = $129
100 = $189
250 = $389
500 = $679
1000 = $1,199

Change the prices object in script.js to match your actual pricing.

3. LOGO UPLOAD
The logo upload currently previews the customer's logo locally in their browser.
It does not transmit or store the file.

4. ORDER FORM
The order form opens the customer's email application.
For direct submissions, connect Formspree, Basin, Netlify Forms, or a serverless backend.

5. STRIPE
GitHub Pages is static hosting and cannot securely store Stripe secret keys.
Safe options:
- Stripe Payment Links
- Stripe Checkout through a serverless backend
- Move the site to Vercel/Netlify and add serverless functions

6. PHOTOS
The gallery currently uses styled product mockups.
Replace these with actual product photography as soon as you have high-quality examples.

7. SEO
robots.txt, sitemap.xml, canonical URL, Open Graph data, and Organization schema are included.
Submit the sitemap to Google Search Console after launch.
